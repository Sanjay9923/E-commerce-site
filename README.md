# E-commerce-site

## Overview

Glowing is a responsive eCommerce website that focuses on a clean layout and smooth user experience. It adapts well to all screen sizes and loads fast on both mobile and desktop. The design is simple, the codebase is easy to understand and the project is suitable for anyone looking to study how an online store interface is built.

## Features

- Fully responsive layout that works on mobile, tablet and desktop

- Clean product grid with hover effects

- Simple and clear navigation bar

- Smooth scrolling for better user experience

- Minimal and fast loading design

- Organized code that is easy to customize

- Supports product images and basic product display

## Getting Started

To run **Glowing** locally, run this command on your git bash:

1.**Clone the Repository**

Linux / macOS

git clone https://github.com/Sanjay9923/E-commerce-site.git

Windows

git clone https://github.com/Sanjay9923/E-commerce-site.git

2.**Open the Project Folder**

cd E-commerce-site

3.**Run the Website**

Just open the file below in your browser:

index.html

That’s it. The website will load immediately
